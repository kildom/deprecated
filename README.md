 Program konsolowy dla systemu Windows. Umożliwia otwieranie plików i programów z wiersza poleceń z nieco innymi możliwościami niż standardowe polecenie "START".

Wszystko co robi ten program, to uruchamia funkcję systemu Windows ShellExecute przekazując parametry otrzymane w wierszu poleceń do niej. Umożliwia to m.in. zmianę katalogu startowego, sposób wyświetlenia okna lub nawet jego ukrycie, zmiany operacji na inną niż domyślna np. drukowanie pliku zamiast zwykłego otwarcia.
